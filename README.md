## ğŸŒ Overview

**FarmConnect** is a comprehensive digital agriculture platform built with Django that bridges the gap between Kenyan small-scale farmers and market opportunities. Unlike traditional e-commerce platforms, FarmConnect focuses on **empowering farmers** through direct farm-to-market connections, agricultural education, community engagement, and AI-powered advisory services.

Our mission is to eliminate middlemen, provide fair pricing, offer agricultural knowledge, and create a sustainable digital ecosystem where farmers can thrive economically while practicing environmentally responsible agriculture.

## ğŸ¯ Problem Statement

Small-scale farmers in Kenya face numerous challenges:
- **Limited Market Access**: Dependence on middlemen who reduce profit margins
- **Knowledge Gap**: Lack of access to modern agricultural practices and techniques
- **Price Exploitation**: Unfair pricing due to lack of market information
- **Isolation**: Limited farmer-to-farmer communication and knowledge sharing
- **Environmental Concerns**: Need for sustainable and green farming practices
- **Technology Barrier**: Difficulty adopting digital tools without guidance

**FarmConnect** addresses these challenges by providing:
- **Direct Sales Platform**: Sell directly from your farm to buyers
- **FarmAI Chatbot**: 24/7 AI-powered agricultural advisor on green economy and sustainable practices
- **Community Hub**: Connect, share experiences, and learn from fellow farmers
- **Market Intelligence**: Real-time price information and demand forecasts
- **Educational Resources**: Training materials, videos, and best practices
- **Sustainable Agriculture Focus**: Promoting eco-friendly farming methods

## ğŸš€ Installation & Setup

### Prerequisites
- Python 3.10 or higher
- PostgreSQL (recommended) or SQLite
- Redis (for caching and Celery)
- Git
- OpenAI API key (for FarmAI)

### Step-by-Step Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/farmconnect.git
   cd farmconnect
   ```

2. **Create Virtual Environment**
   ```bash
   python -m venv venv
   
   # On Windows
   venv\Scripts\activate
   
   # On macOS/Linux
   source venv/bin/activate
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Environment Configuration**
   
   Create `.env` file in project root:
   ```env
   # Django Settings
   SECRET_KEY=your-secret-key-here
   DEBUG=True
   ALLOWED_HOSTS=localhost,127.0.0.1
   
   # Database
   DATABASE_URL=postgresql://user:password@localhost:5432/farmconnect
   # Or for SQLite: DATABASE_URL=sqlite:///db.sqlite3
   
   # FarmAI Configuration
   OPENAI_API_KEY=your-openai-api-key
   FARMAI_MODEL=gpt-4
   FARMAI_MAX_TOKENS=1000
   
   # WhatsApp Business API
   WHATSAPP_BUSINESS_NUMBER=254XXXXXXXXX
   WHATSAPP_API_TOKEN=your-whatsapp-token
   
   # M-Pesa Configuration
   MPESA_CONSUMER_KEY=your-mpesa-consumer-key
   MPESA_CONSUMER_SECRET=your-mpesa-consumer-secret
   MPESA_SHORTCODE=your-business-shortcode
   MPESA_PASSKEY=your-mpesa-passkey
   MPESA_CALLBACK_URL=https://yourdomain.com/api/mpesa/callback/
   
   # Weather API
   WEATHER_API_KEY=your-weather-api-key
   
   # Email Configuration
   EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
   EMAIL_HOST=smtp.gmail.com
   EMAIL_PORT=587
   EMAIL_USE_TLS=True
   EMAIL_HOST_USER=your-email@gmail.com
   EMAIL_HOST_PASSWORD=your-app-password
   
   # SMS Gateway (Optional)
   SMS_API_KEY=your-sms-api-key
   SMS_SENDER_ID=FARMCONNECT
   
   # Redis (for Celery and caching)
   REDIS_URL=redis://localhost:6379/0
   
   # Media and Static
   MEDIA_URL=/media/
   STATIC_URL=/static/
   ```

5. **Database Setup**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

6. **Create Superuser**
   ```bash
   python manage.py createsuperuser
   ```

7. **Load Sample Data**
   ```bash
   python manage.py seed_data
   ```
   
   This will create:
   - Sample farmers and farms
   - Product categories and listings
   - Counties, sub-counties, and wards
   - Educational content
   - Community posts
   - Market prices

8. **Create Required Directories**
   ```bash
   mkdir -p logs media/farms media/products media/community media/education
   ```

9. **Collect Static Files**
   ```bash
   python manage.py collectstatic
   ```

10. **Start Redis (for FarmAI and Celery)**
    ```bash
    redis-server
    ```

11. **Start Celery Worker (in new terminal)**
    ```bash
    celery -A FarmConnect worker -l info
    ```

12. **Run Development Server**
    ```bash
    python manage.py runserver
    ```

13. **Access the Platform**
    - Homepage: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
    - Admin Panel: [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)
    - FarmAI Chat: [http://127.0.0.1:8000/farmai/](http://127.0.0.1:8000/farmai/)
    - Community: [http://127.0.0.1:8000/community/](http://127.0.0.1:8000/community/)

```python
# apps/payments/mpesa.py
from daraja import Mpesa

def initiate_payment(phone, amount, account_ref):
    mpesa = Mpesa()
    response = mpesa.stk_push(
        phone_number=phone,
        amount=amount,
        account_reference=account_ref,
        transaction_desc="Farm Product Purchase"
    )
    return response
```
